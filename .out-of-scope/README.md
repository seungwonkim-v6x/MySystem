# Out of Scope

Decisions to **not do** something, with the reasoning. This directory exists so
future-me (or another contributor) doesn't have to re-derive every "we
considered it and chose no" decision from scratch.

Rules:

- One file per decision. Filename describes what was rejected.
- Body: what was considered, why it was attractive, why we rejected it, what
  would make us reconsider.
- Reopen a decision by deleting the file and writing a new ADR or CHANGELOG
  entry that supersedes it.

Inspired by [mattpocock/skills/.out-of-scope](https://github.com/mattpocock/skills/tree/main/.out-of-scope).
